プロジェクトの説明

県警プロジェクトは、福岡工業大学システムマネジメント学科３年生が実施します。公開データを基にAIやシミュレーションシステム、可視化システムなどの最新テクノロジーを活用して福岡県の交通システムの課題を抽出し、解決策を提案することを目的としています。システムは、AWS アカデミーのサポートを受けてクラウドでホストされます。

目的

1. 公開データを使用して、時間と曜日、場所、道路の種類などの潜在的な事故状況を把握します。(AI)
2. さまざまな交通状況の実際の道路をシミュレーションをSUMO シミュレーターで行います。そして交通渋滞をより深く理解し、渋滞を軽減するためのアイデアを提案します。
3. 事故現場や交通状況を可視化することで、危険な道路での運転や渋滞の回避などに役立ちます。(unity)

実施期間

2024年4月～2025年1月

活動内容

1. さまざまなソースからアイデアとオープンデータを収集します。
2. 導入のためのAWSクラウド環境を構築します。
3. プロジェクトの Web ページを作成します。
4. データを準備してデータベースにアップロードします。
5. データをテストします。
6. 問題のある交差点や渋滞のシナリオをテストするためのシミュレーションシステムを作成します。
7. Unity 上でデータを視覚化するか、Web ページでマップを表示します。

メンバー

福岡工業大学 情報工学部 システムマネジメント学科の3年生6名、4年生2名。
一時的な追加メンバーを歓迎します。

学習目標

1. 機械学習システムを設計できること。
2. AWS でクラウド ソリューションを実装できること。
3. SUMO シミュレータでさまざまなシミュレーション シナリオを構築して実行できる。
4. さまざまなデザインでデータを視覚化できる。

スキル

AWSクラウドインフラストラクチャ, 視覚化, SUMOシミュレーター, 最適化, ウェブデザイン
